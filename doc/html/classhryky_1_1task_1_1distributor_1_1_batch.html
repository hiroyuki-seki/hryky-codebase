bool Base::attach ( ) [inherited]

attaches all workers to concurrent units.

Returns:
success or failure.
  • If the workers are not attached to each concurrent unit, they are added to the waiting queue of the each concurrent unit synchronously.
Attention:
  • This function includes the synchronization using exclusive control. (i.e. The calling context may be suspended for log periods.)
